Tested with Graphviz version 2.24.

Install graphviz and include GraphViz\bin into PATH or just copy graphviz dll's into project output directory, required set is:
cdt.dll
graph.dll
gvc.dll
gvcplugin_core.dll
gvcplugin_dot_layout.dll
libexpat.dll
ltdl.dll
Pathplan.dll
regex_win32.dll
and with dll's required configuration file "config6"(see config6.sample)

GraphVizNet is Visual C++ 2008 project, if you don't have installed Visual C++ see precompiled version of GraphVizNet.dll in "precompiled" folder.
